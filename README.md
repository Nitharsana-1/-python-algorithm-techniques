# Algorithm Design Techniques — Python Implementations

Implementation of classic algorithms covering Backtracking, 
Brute Force, and Divide & Conquer techniques in Python.

## Algorithms Implemented

| File | Algorithm | Technique |
|------|-----------|-----------|
| permutations.py | Generate all permutations | Backtracking |
| combinations.py | Generate all combinations | Backtracking |
| tsp_brute_force.py | Travelling Salesman Problem | Brute Force |
| knapsack_brute_force.py | 0/1 Knapsack Problem | Brute Force |
| knapsack_combinations.py | Knapsack via Combinations | Backtracking |
| closest_pair_divide_conquer.py | Closest Pair of Points | Divide & Conquer |
| karatsuba_multiplication.py | Karatsuba Fast Multiplication | Divide & Conquer |
| matrix_multiplication.py | Matrix Multiplication | Arrays |
| binary_bit_check.py | Binary Representation & Bit Check | Bit Manipulation |
| brute_force_otp.py | OTP Brute Force Simulation | Brute Force |

## Key Highlights
- **TSP**: Finds optimal route across 5 cities using permutation-based brute force
- **Knapsack**: Solves 0/1 knapsack with both brute force and combination approaches
- **Closest Pair**: Divide & conquer implementation with detailed step-by-step trace
- **Karatsuba**: Fast multiplication algorithm reducing O(n²) to O(n^1.585)

## Techniques Covered
- Backtracking
- Brute Force
- Divide & Conquer
- Bit Manipulation
